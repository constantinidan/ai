# Grok Provider

The [Grok](https://x.ai) provider contains language model support for the [Grok API](https://x.ai/api).

## Setup

The Grok provider is available via the `@ai-sdk/grok` module.
You can install it with

<Tabs items={['pnpm', 'npm', 'yarn']}>
  <Tab>
    <Snippet text="pnpm add @ai-sdk/grok" dark />
  </Tab>
  <Tab>
    <Snippet text="npm install @ai-sdk/grok" dark />
  </Tab>
  <Tab>
    <Snippet text="yarn add @ai-sdk/grok" dark />
  </Tab>
</Tabs>

## Provider Instance

You can import the default provider instance `grok` from `@ai-sdk/grok`:

```ts
import { grok } from '@ai-sdk/grok';
```

If you need a customized setup, you can import `createGrok` from `@ai-sdk/grok`
and create a provider instance with your settings:

```ts
import { createGrok } from '@ai-sdk/grok';

const grok = createGrok({
  // custom settings
});
```

You can use the following optional settings to customize the Grok provider instance:

- **baseURL** _string_

  Use a different URL prefix for API calls, e.g. to use proxy servers.
  The default prefix is `https://api.x.ai/v1`.

- **apiKey** _string_

  API key that is being send using the `Authorization` header.
  It defaults to the `GROK_API_KEY` environment variable.

- **headers** _Record&lt;string,string&gt;_

  Custom headers to include in the requests.

- **fetch** _(input: RequestInfo, init?: RequestInit) => Promise&lt;Response&gt;_

  Custom [fetch](https://developer.mozilla.org/en-US/docs/Web/API/fetch) implementation.
  Defaults to the global `fetch` function.
  You can use it as a middleware to intercept requests,
  or to provide a custom fetch implementation for e.g. testing.

## Language Models

You can create [Grok models](https://console.x.ai) using a provider instance.
The first argument is the model id, e.g. `grok-beta`.

```ts
const model = grok('grok-beta');
```

### Example

You can use Grok language models to generate text with the `generateText` function:

```ts
import { grok } from '@ai-sdk/grok';
import { generateText } from 'ai';

const { text } = await generateText({
  model: grok('grok-beta'),
  prompt: 'Write a vegetarian lasagna recipe for 4 people.',
});
```
